# 仓库信息

本项目同时推送到以下远程仓库：

| 名称 | 地址 | 平台 |
|------|------|------|
| origin | `git@github.com:borisxxz/CipherTalk_borisxxz.git` | GitHub |
| cnb | `https://cnb.cool/boris007/CipherTalk_self.git` | CNB |

## 推送命令

```bash
# 推送到 GitHub（origin）
git push origin main

# 推送到 CNB
git push cnb main

# 同时推送两个
git push origin main && git push cnb main
```

## 备注

- `myrepo` 和 `origin` 指向同一个 GitHub 仓库，`myrepo` 可删除
- 原始上游仓库: `https://github.com/ILoveBingLu/CipherTalk`（已移除 upstream）
